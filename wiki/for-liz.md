## Chat with the wiki

You can ask this wiki questions directly, without needing to dig through pages.

Launch the chat: say `/run-wiki-chat` in Claude Code. A local page opens where you can type questions like:

- _"What's ready for me to test?"_
- _"What shipped this week?"_
- _"Remind me how rent reminders work."_
- _"What did we decide about tenant screening?"_

The chat answers using only what's written in this wiki — it won't make things up, and if it doesn't know something, it'll tell you.

> The chat feature is being built in the current development round (P1-Tkt-002). Once it's live, you'll see it in the project status above.
