# Product Owner Feedback → 4 New Phase 1 Features (P1-004 through P1-007)

## Decision

Liz's review of deployed onboarding and dashboard generated four new Phase 1 feature areas: onboarding UX refinements (P1-004), property-centric dashboard (P1-005), lease/document management (P1-006), and utility company integration (P1-007). Bug T-011 (slider) also logged.

## Rationale

Direct product owner feedback on deployed software is the highest-signal input for product decisions. Organizing the feedback into four named features rather than a flat bug list preserves the thematic grouping and makes it easier to plan and execute. Each feature maps to a coherent user problem rather than an isolated UI complaint.

## Consequences

- Features P1-004 through P1-007 are added to the Phase 1 roadmap.
- These are ticket-driven features (Tkt convention), not pre-planned roadmap items.
- All four features are later consolidated into `P1-Tkt-001-mvp-ux-overhaul` (see related decision).
