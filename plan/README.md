## Task Management

Tasks live inside their feature directory under `features/inprogress/<feature>/`:

```
features/inprogress/<feature>/
├── README.md    ← feature plan
├── backlog/     ← pending tasks
├── doing/       ← current task
└── done/        ← completed tasks
```

See `features/inprogress/` for active features and `features/completed/` for finished ones.
